Sistem Informasi Penggajian Karyawan Berbasis Web

Aplikasi Sistem Informasi Penggajian Berbasis Web. Dibuat dengan bahasa pemrograman PHP dan Framework CodeIgniter.
Aplikasi ini memiliki fitur untuk Admin yaitu CRUD Data Karyawan, CRUD Data Jabatan, Input Data Kehadiran, CRUD Potongan Gaji, Melihat Gaji Karyawan Berdasarkan Bulan dan Tahun yang dipilih, Cetak Gaji Karyawan Berdasarkan Bulan dan Tahun, Cetak Slip Gaji Karyawan Berdasarkan Nama Karyawan, Bulan, dan Tahun yang dipilih.
Dan fitur untuk Karyawan yaitu Hanya dapat melihat Data Karyawan, Data Kehadiran, dan Data Gaji.

Toko Online Berbasis Web

Aplikasi ini memiliki fitur untuk hak akses Admin yaitu CRUD Data Barang dan Melihat Invoice.
Dan fitur untuk hak akses pembeli yaitu Melihat detail barang, Memasukkan barang kedalam keranjang belanja dan Melakukan pembayaran pesanan.
